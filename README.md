## About

This repository documents my personal journey learning database languages — from writing my first SQL queries to exploring NoSQL with MongoDB.

Each folder represents a learning stage with real `.sql` scripts, exercises, and notes I wrote while studying. If you're just starting out with databases, feel free to follow along.

---

## What You'll Learn

- **DDL** — `CREATE`, `ALTER`, `DROP` (Data Definition Language)
- **DML** — `INSERT`, `UPDATE`, `DELETE` (Data Manipulation Language)
- **DQL** — `SELECT` queries from simple to complex (joins, subqueries, aggregations)
- **T-SQL** — SQL Server–specific syntax and stored procedures
- **MongoDB** — document model, collections, CRUD operations in NoSQL

## How to Use

1. Clone the repo:
```bash
git clone https://github.com/JoshTVR/DataBaseLanguageForBeginers.git
```

2. Open any `.sql` file in **SQL Server Management Studio (SSMS)** or **Azure Data Studio**
3. For MongoDB scripts, use **MongoDB Compass** or the `mongosh` CLI
